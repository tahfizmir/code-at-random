Tech stack used

React frontend , Node.js/express.js backend

How to run backend

if running locally:
clone the repo and run the following commands
cd backendfolder -> npm install -> npm run start

How to run frontend

if running locally:
clone the repo and run the following commands
cd frontendfolder -> npm install -> npm run dev -> navigate to http://localhost:5173

API endpoints

post('/skill-gap')
post('/roadmap')
get('/news')

Any assumptions

    The server may take up to 50 seconds to respond on your first request
    because it’s hosted on a free Render instance that “sleeps” when idle.
    After the first response, everything will run fast and smoothly.
    Thanks for your patience!
